# Project Preparation

## What is the one core feature your project must have to work?

The most important feature is Global Horizontal Irradiance (GHI), since solar power generation is primarily driven by incoming solar radiation.

## What dataset will you be using?

A real-world photovoltaic power generation dataset containing:

- Temperature (°C)
- Global Horizontal Irradiance (W/m²)
- Wind Speed (m/s)
- Precipitation (L/m²)
- Active Power (kW)
- Reactive Power (kVar)

The dataset consists of 10-minute measurements collected from a photovoltaic power plant.
